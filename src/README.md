# Sum of Digits Calculator

A modern, fast, and interactive application built with **React** and **Vite** that calculates the sum of digits for any given number.

## Features

- **Real-time Calculation:** Instantly calculates the sum as you type.
- **Visual Feedback:** Displays the individual digits being summed.
- **Modern UI:** Dark-themed, glass-morphism design using pure CSS (no external UI frameworks).
- **Vite Powered:** Fast development and optimized build.

## Project Structure

This project was migrated from a legacy Create-React-App structure to Vite.

- `src/App.jsx`: Main application logic.
- `src/App.css`: Custom styling.
- `src/index.html`: Entry point.

## Getting Started

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Run Development Server:**
    ```bash
    npm run dev
    ```

3.  **Build for Production:**
    ```bash
    npm run build
    ```
